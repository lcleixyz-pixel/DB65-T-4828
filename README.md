## 一、超景深显微镜期间核查

**适用设备**：海康威视 DI-AM1160 型超景深显微镜  
**依据标准**：GB/T 24762-2009、JJF 1318-2011 及所内《期间核查规程》

### 快速使用

1. 用 Chrome 或 Edge 打开：`超景深显微镜/#超景深显微镜期间核查工具.html`
2. 按左侧 9 步向导录入数据（可先点「加载所有演示数据」体验）
3. 在步骤 9 生成 3 页 A4 报告，打印或另存为 PDF

### 核查项目（7 项）

| 代号 | 内容 | 判定要点（摘要） |
|------|------|------------------|
| — | 外观与功能性 | 载物台、镜头、光源、软件拼接与测量 |
| EXY | 平面尺寸测量误差 | 示值误差 ≤±1%×L，重复性 ≤±0.33%×L |
| EZ | 垂直尺寸测量误差 | 同上 |
| EV | 影像测头尺寸测量误差 | 同上 |
| PV | 影像测头探测误差 | ≤10 μm（参考） |
| P2D | 二维探测误差 | ≤10 μm（参考） |
| PZ | 变倍探测误差 | ≤10 μm（参考） |

**注意**：数据仅存于浏览器内存，刷新页面会丢失；完成后请立即生成并保存报告。需联网加载 CDN（Tailwind、Chart.js 等）。

详细操作见 `超景深显微镜/0.1使用说明.txt`，方法与步骤见 `1.3期间核查规程.md`。

## 二、紫外–可见光谱导数分析

**用途**：对光谱仪导出的谱图进行平滑、基线校正、一/二阶导数、极值检测与可视化；适用于宝石鉴定、和田玉皮色鉴别等。

### 完整版（推荐）

路径：`紫外-可见光谱/UV-Vis 光谱导数分析工具/`

**环境**：Windows，Python 3.8+

```text
双击「启动应用.bat」
  → 自动创建 .venv、安装 requirements.txt
  → 浏览器打开 http://localhost:8501
```

也可先运行 `安装依赖.bat` 仅安装环境。命令行方式见该目录 [README.md](紫外-可见光谱/UV-Vis%20光谱导数分析工具/README.md)。

**主要能力**

- 上传 `.txt` / `.csv`（波长 + 强度两列）
- Savitzky-Golay 平滑、ALS 基线校正、波长裁剪
- 一/二阶导数（独立窗口，支持降序波长）
- 自动峰/谷检测与单图点击手动标记
- 透射率 ↔ 吸光度、反射率 ↔ Kubelka-Munk 转换
- 参数预设（`presets.json`）、和田玉参考线（435/535/575 nm）
- 导出 Excel、PNG、HTML

**文档**（均在完整版目录内）

| 文档 | 内容 |
|------|------|
| 安装配置引导.md | 新环境安装 Python 与依赖 |
| 用户手册.md | 数据格式、界面与操作 |
| 和田玉皮色鉴别指南.md | 漫反射光谱鉴别皮色 |
| 数值分析实践及工具的实用性分析.md | 算法原理与实践 |

### 简化版（免安装）

路径：`紫外-可见光谱/UV-Vis 光谱导数分析工具（简）.html`

浏览器打开即可：Plotly 绘图 + 客户端 SG 平滑/导数，可导出 CSV。功能少于 Streamlit 版，适合无 Python 环境时快速查看。

## 版本管理与协作

本仓库已配置 Git。首次克隆或拉取后按各子工具说明使用即可。

```bash
# 查看状态
git status

# 查看历史
git log --oneline

# 关联远程并推送（需自行创建空仓库后替换 URL）
git remote add origin <你的仓库 URL>
git push -u origin main
```

`.gitignore` 已排除 `.venv`、`__pycache__` 及本地导出的 xlsx 等；`presets.json` 会纳入版本库。
