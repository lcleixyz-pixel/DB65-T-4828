平面尺寸测量误差 (EXY)

操作要求

  • 标准器：0-20mm 玻璃刻度尺（分度 0.1mm）
  • 倍率设置：50X（固定）
  • X轴方向：使用 2D 拼接模式（手动 5×1 拼接），沿X轴拼接完整影像
  • Y轴方向：将刻度尺旋转 90°，使用 2D 拼接模式（手动 1×7 拼接），沿Y轴拼接
  • 测量方法：使用"垂基线"功能，测量5个刻度位置（4mm, 8mm, 12mm, 16mm, 20mm）
  • 重复性：每个间隔重复测量 3 次
单位说明：仪器输出单位为 μm，可直接输入（如 4000、8000）；工具将自动转换为 mm 进行计算。标准值已预设为 mm（4、8、12、16、20）。

垂直尺寸测量误差 (EZ)

操作要求

  • 标准器：4mm 量块、5mm 量块（形成 1mm 高度差台阶）
  • 倍率设置：推荐 300X（3D 模式）
  • 拼接方法：使用 3D 拼接功能，以高度差台阶为视野中心进行 3×1 高精度拼接
  • 测量方法：将 4mm 量块表面设为基准面，对 5mm 量块表面测量 3 个点（中心 + 2 端点）
  • 重复性：重复拼接并测量 3 次
单位说明：仪器输出单位为 μm，可直接输入（标准值 1mm = 1000μm）；工具将自动转换为 mm 进行计算。

影像测头尺寸测量误差 (EV)

操作要求

  • 说明：仅利用视场（FOV）测量，禁止移动载物台
  • 标准器：0-1mm 玻璃刻度尺（分度 0.01mm）
  • 倍率设置：300X
  • 测量方法：使用"垂基线"功能，测量覆盖视场约 2/3 的刻度（如 0.6mm 或 0.8mm）
  • 测量位置：改变刻度尺在视场中的位置（上、中、下），各测量 3 个位置
单位说明：标准值输入 mm（如 0.6、0.8）；仪器输出单位为 μm，可直接输入（如 600、800）；工具将自动转换为 mm 进行计算。

影像测头探测误差 (PV)

操作要求

  • 说明：仅借助视场开展测量，禁止移动载物台，以评估镜头畸变
  • 标准器：圆形靶标
  • 倍率设置：80X（使圆的大小约占视场 30%—60%）
  • 采点方法：运用软件的"XY测量"功能，使 X/Y 轴与圆的下缘/左缘近乎相切，将测量点依次放置在圆周上均匀分布的 15 个位置（需保证采集点覆盖整个圆）
  • 拟合：由软件计算最小二乘圆
单位说明：X/Y 坐标值直接从仪器软件读取并输入（单位：μm），无需换算。结果 PV 单位为 μm。

二维探测误差 (P2D)

操作要求

  • 说明：需要联动载物台进行测量，以评估综合精度
  • 标准器：圆形靶标
  • 倍率设置：300X
  • 拼接方法：使用 2D 拼接模式，以圆心为视野中心进行 4×3 拼接，得到圆靶标的完整影像
  • 采点方法：运用软件的"XY测量"功能，使 X/Y 轴与圆的下缘/左缘近乎相切，将测量点依次放置在圆周上均匀分布的 25 个位置（需保证采集点覆盖整个圆）
  • 拟合：采用最小二乘法拟合圆形
单位说明：X/Y 坐标值直接从仪器软件读取并输入（单位：μm），无需换算。结果 P2D 单位为 μm。

变倍探测误差 (PZ)

操作要求

  • 标准器：0-1mm 玻璃刻度尺（分度 0.01mm）
  • 变倍序列：200X → 150X → 100X
  • X轴测量：在 200X 倍率下，使十字形标尺的纵线与刻度尺零刻度线左侧外缘重合，记录 X 坐标；分别切换至 150X、100X，重复对齐并记录
  • Y轴测量：切回 200X，旋转载物台 90°，使十字形标尺的横线与刻度尺零刻度线上侧外缘重合，记录 Y 坐标；重复变倍操作
单位说明：X/Y 坐标值直接从仪器软件读取并输入（单位：μm），无需换算。结果 PZ 单位为 μm。
