# 光谱导数分析工具 — 用户手册

本工具用于对紫外–可见光谱仪导出的谱图进行本地分析，提供平滑、基线校正、一/二阶导数计算、自动极值检测及多视图可视化，适用于宝石鉴定、和田玉皮色鉴别等场景。

**快速启动**：双击 `启动应用.bat`，浏览器自动打开。首次运行会自动检测并安装依赖；也可先双击 `安装依赖.bat` 完成环境配置后再启动。

## 1. 数据准备与导入

### 1.1 环境要求
- Python 3.8+
- 依赖安装：首次运行 `启动应用.bat` 会自动安装；或先运行 `安装依赖.bat`；或手动执行 `.venv\Scripts\python -m pip install -r requirements.txt`

### 1.2 启动方式
- **推荐**：双击 `启动应用.bat`（Windows），首次会自动安装依赖后启动
- **命令行**：在项目目录执行 `.venv\Scripts\python -m streamlit run app.py --server.port 8501`  
启动后浏览器自动打开 `http://localhost:8501`。

### 1.3 数据格式要求
- **文件类型**：`.txt` 或 `.csv` 文本文件（不支持 .xlsx）
- **数据结构**：至少两列数值  
  - 第一列：波长 (nm)  
  - 第二列：强度（透射率、吸光度或反射率，0–1 或 0–100%）
- **兼容性**：自动跳过表头与注释行（`#` 开头），自动识别分隔符与编码（UTF-8、GBK 等）。

**示例**：
```text
380.0   0.002
380.5   0.003
381.0   0.005
...
```

### 1.4 导入步骤
1. 点击左侧 **「上传数据文件」** 或主界面的 **「Browse files」**
2. 选择 `.txt` 或 `.csv` 文件
3. 主界面出现图表即表示导入成功；若报错，请检查文件格式与列数

## 2. 界面与功能概览

### 2.1 主界面
- **顶部**：当前文件名、数据点数、波长范围、数据类型
- **折叠面板**：波长范围截取、数据类型转换
- **主图表**：根据视图模式显示原始谱、预处理谱、一阶/二阶导数（四图总览 / 合并视图 / 单图放大）

### 2.2 侧边栏结构
- **参数预设**：选择/加载/保存预设（如「默认配置」「和田玉K-M标准」）
- **视图模式**：四图总览、合并视图、单图放大
- **标签页**：
  - **⚙️ 处理**：平滑、基线校正、导数参数
  - **🔍 检测**：自动极值检测（峰/谷、数据源、显著性、最小间距）
  - **📍 标记**：手动标记、参考线（含「快速添加和田玉参考线」）
  - **💾 导出**：Excel、PNG、HTML

### 2.3 工具能做什么
- **去噪**：Savitzky–Golay 平滑，保留峰形
- **拉直基线**：ALS 基线校正，减轻漂移
- **锐化特征**：一阶/二阶导数，突出肩峰与重叠峰
- **自动寻峰**：按峰/谷模式自动标记极值
- **点击取点**：单图模式下点击图表自动填入波长与强度，便于手动标记
- **参考线**：添加固定波长垂线，便于与标准峰位对照

## 3. 操作流程（通用）

以「看清特征峰」为例（如红宝石 694 nm Cr 峰）：

1. **启动与导入**：运行 `启动应用.bat`，上传光谱文件。
2. **确认数据类型**：看纵坐标是透射率还是吸光度。若为透射率且希望峰向上，在主界面展开「🔄 数据类型转换」，选「转换为：吸光度」并执行转换。
3. **预处理**：侧边栏「⚙️ 处理」中调节「平滑窗口」（一般 11–21）、勾选「启用基线校正」、调节「基线 λ」（默认 5.0 可先不动）。
4. **导数分析**：查看主界面下方一阶/二阶导数图。若二阶导数噪声大，将「二阶导数窗口」调大（如 41–51）。
5. **标记与参考线**：视图模式选「单图放大」，选择「一阶导数」或「二阶导数」，在图上点击峰顶，侧栏「📍 标记」中补全说明后点击「➕ 添加标记」；需要时可输入波长添加参考线。
6. **导出**：在「💾 导出」标签页下载 Excel（数据）、PNG（插图）或 HTML（交互报告）。

## 4. 数据类型转换

### 4.1 支持的类型
- **透射率 (0–1)**、**透射率 (%)**：透过光强/入射光强
- **吸光度**：A = −log₁₀(T)，适用于 Beer–Lambert 定量分析
- **反射率 (0–1)**、**反射率 (%)**：反射光强/入射光强，常用于漫反射光谱
- **Kubelka–Munk**：F(R) = (1−R)²/(2R)，用于漫反射样品，使吸收峰更明显

### 4.2 何时需要转换
- **透射光谱**：建议转为**吸光度**再求导，峰更清晰、物理意义明确。
- **漫反射光谱（如和田玉皮色）**：建议转为 **Kubelka–Munk** 再求导，详见《和田玉皮色鉴别指南》。

### 4.3 操作步骤
1. 在主界面展开 **「🔄 数据类型转换」**
2. **当前类型**：选择与实际数据一致（工具会根据数值范围自动推断并预填）
3. **转换为**：选择目标类型（如吸光度、Kubelka–Munk）
4. 点击 **「执行转换」**；若要恢复，点击 **「恢复原始」**

## 5. 参数与预设

### 5.1 处理参数（⚙️ 处理）
- **平滑窗口**：5–51，奇数；越大越平滑，过大可能压扁窄峰。常用 15–21。
- **多项式阶数**：2–5，通常取 3。
- **启用基线校正**：勾选后启用 ALS 基线扣除。
- **基线 λ (10^x)**：3–8，越大基线越平；常用 5.0–6.0。
- **一阶导数窗口**：5–51，常用 21–25。
- **二阶导数窗口**：建议为一阶的 1.5–2 倍（如 41–51），以抑制噪声。

### 5.2 极值检测（🔍 检测）
- **检测数据源**：基线校正后 / 平滑谱 / 原始谱
- **检测模式**：自动（按数据类型选峰或谷）、峰、谷
- **显著性阈值**：相对信号范围的比例，用于过滤弱峰
- **最小间距**：相邻峰/谷最少间隔点数

### 5.3 预设的保存与加载
- **加载**：在侧边栏顶部「参数预设」下拉框选择预设名，点击「加载」。
- **保存**：调好参数后，点击「💾 保存/管理预设」，输入名称后保存。可在此删除不需要的预设（「默认配置」不可删）。

内置预设示例：**和田玉K-M标准**、**和田玉K-M-高分辨**、**和田玉K-M-降噪**，专用于漫反射 K–M 数据。

## 6. 标记与参考线

### 6.1 手动标记
1. 视图模式选「单图放大」，并选好要标记的子图（如一阶导数）。
2. 在图表上点击目标位置，侧栏「📍 标记」中波长、强度会自动填入。
3. 填写「说明」后点击「➕ 添加标记」。可在「已添加 N 个标记」中逐个删除或「清空所有标记」。

### 6.2 参考线
- **单条添加**：在「参考线」区域输入参考波长、标签、颜色，点击「➕ 添加参考线」。
- **快速添加和田玉参考线**：点击「🟡 快速添加和田玉参考线」可一次添加 435 nm、535 nm、575 nm 三条标准峰位线（针铁矿次峰/主峰、赤铁矿峰），用于和田玉皮色分析；重复点击不会重复添加已存在的相近波长线。

### 6.3 参考线管理
在「已添加 N 条参考线」展开项中可逐条删除。

## 7. 导出与报告

在侧边栏「💾 导出」标签页：

- **📊 下载 Excel 数据**：多 Sheet（原始、预处理、含导数、标记点），便于存档与二次分析。
- **🖼️ 下载 PNG**：当前视图的高清图，可插入报告或证书。

文件名会基于当前上传文件名自动生成（如 `xxx_processed.xlsx`、`xxx_spectra.png`）。

## 8. 常见问题

**Q：上传后报错或图是空的？**  
A：确认文件为 .txt 或 .csv（非 .xlsx）；至少两列数值；若为 Excel 导出，请另存为「CSV (逗号分隔)」。

**Q：二阶导数图全是毛刺？**  
A：增大「二阶导数窗口」（如 51 或更大）；若数据为透射率，可先转为吸光度再求导。

**Q：图表卡顿？**  
A：数据点过多（如数万点）时浏览器压力大，可降低仪器导出分辨率或先裁剪波长范围后再分析。

**Q：基线校正后出现负值？**  
A：工具会对负值做裁剪并可能给出提示。对透射率/反射率建议先转为吸光度或 K–M 再做基线校正，或适当降低基线 λ。

**Q：如何快速复现上次分析？**  
A：使用「参数预设」：调好一次后保存为预设，下次同类型样品先选预设再点「加载」。

更多针对**和田玉皮色鉴别**的操作与判定，请参阅《和田玉皮色鉴别指南》；算法原理与课程实践见《数值分析实践及工具的实用性分析》。
