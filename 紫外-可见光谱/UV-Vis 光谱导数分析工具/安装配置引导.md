# UV-Vis 光谱导数分析工具 — 安装配置引导（小白版）

本文档面向从未配置过 Python 环境的用户，在新电脑上从零完成安装与首次运行。按顺序执行即可。

---

## 一、你需要准备的东西

- **项目文件夹**：整个 `UV-Vis 光谱导数分析工具` 文件夹（或上级 `紫外-可见光谱` 文件夹）已复制到新电脑，路径中**不要**包含过多空格或特殊符号，推荐例如：
  - `D:\紫外-可见光谱\UV-Vis 光谱导数分析工具`
  - `C:\Tools\UV-Vis 光谱导数分析工具`
- **（可选）项目自带的 Python 安装包**：若提供方在项目中放了 Python 安装程序，一般在**上一级目录**的 `python` 文件夹里，例如：
  - `紫外-可见光谱\python\` 下有 `python-3.x.x-amd64.exe` 或 `python-manager-25.2.msix` 等文件。

---

## 二、第一步：安装 Python

本工具需要 **Python 3.8 或更高版本**。只需安装一次。

### 方案 A：使用项目自带的安装包（推荐，无需联网下载）

1. 打开项目**上一级**目录下的 `python` 文件夹（若没有该文件夹，改用方案 B）。
2. 在文件夹内找到安装程序：
   - **`.exe` 文件**（如 `python-3.12.x-amd64.exe`）：  
     双击运行 → 在安装界面**务必勾选**「**Add python.exe to PATH**」→ 点击「Install Now」完成安装。
   - **`.msix` 文件**（如 `python-manager-25.2.msix`）：  
     这是微软商店格式。双击可能无法直接安装时，请以**管理员身份**打开 PowerShell，执行：
     ```powershell
     Add-AppxPackage -Path "这里改成.msix文件的完整路径"
     ```
     安装完成后，在「开始」菜单中查找「Python 3.x」并运行一次，确认能打开。若命令行仍提示找不到 `python`，请改用方案 B 安装 python.org 的 .exe 版本（PATH 更可靠）。
3. 安装完成后**关闭并重新打开**所有已打开的命令行窗口或资源管理器，再继续下一步。

### 方案 B：从官网下载安装（未提供安装包或方案 A 失败时）

1. 打开浏览器访问：<https://www.python.org/downloads/>
2. 点击页面上的 **「Download Python 3.x.x」**（选 Windows 64 位即可）。
3. 运行下载好的 `.exe`，在**第一个界面**务必勾选：
   - **「Add python.exe to PATH」**
4. 然后点击 **「Install Now」**，按提示完成安装。
5. 安装完成后**关闭并重新打开**所有命令行窗口。

### 验证 Python 是否安装成功

1. 按 `Win + R`，输入 `cmd`，回车，打开命令提示符。
2. 输入：
   ```text
   python --version
   ```
3. 若显示类似 `Python 3.12.x` 则说明安装成功且已加入 PATH。  
   若提示「不是内部或外部命令」，说明 PATH 未生效：
   - 确认安装时勾选了「Add python.exe to PATH」；
   - 重启电脑后再试；
   - 或卸载后重新安装并勾选该项。

## 三、第二步（可选）：预先安装项目依赖

从当前版本开始，`启动应用.bat` 会在首次运行时自动创建 `.venv` 并安装依赖。  
如果你希望先把环境装好再启动，也可以单独执行本步骤。

1. 进入项目目录：在资源管理器中打开 **`UV-Vis 光谱导数分析工具`** 文件夹（即包含 `app.py`、`requirements.txt` 的目录）。
2. **双击运行** **`安装依赖.bat`**。
3. 等待窗口中出现「Setup complete」或「安装完成」类提示；若出现报错，请看下文「故障排除」。

**可选（熟悉命令行的用户）**：在同一目录打开命令提示符，执行：
```text
python -m venv .venv
.venv\Scripts\python -m pip install --upgrade pip
.venv\Scripts\python -m pip install -r requirements.txt
```

---

## 四、第三步：启动应用

1. 仍在 **`UV-Vis 光谱导数分析工具`** 目录下，**双击** **`启动应用.bat`**。
2. 脚本会自动完成检查：
   - 若 `.venv` 不存在，会自动创建项目本地虚拟环境；
   - 若依赖缺失，会自动安装 `requirements.txt`；
   - 安装失败时会自动尝试国内镜像源重试。
3. 浏览器一般会自动打开 **http://localhost:8501**；若未打开，请手动在浏览器地址栏输入该地址。

关闭启动应用时打开的黑框窗口，即可停止程序。

## 五、故障排除

### 1. 提示「Python 不是内部或外部命令」

- **原因**：Python 未加入系统 PATH，或安装后未重启命令行。
- **处理**：
  - 确认安装时勾选了「Add python.exe to PATH」；
  - 关闭所有 cmd/PowerShell 窗口后重新打开，再试 `python --version`；
  - 若仍不行，重启电脑；
  - 若仍不行，卸载 Python 后重新安装，务必勾选「Add python.exe to PATH」。

### 2. 运行「安装依赖」或「启动应用」时报错（如 pip、SSL、Permission denied）

- **网络/超时**：先升级 pip 再安装依赖：
  ```text
  .venv\Scripts\python -m pip install --upgrade pip
  .venv\Scripts\python -m pip install -r requirements.txt
  ```
  若下载很慢，可使用国内镜像（在项目目录下执行）：
  ```text
  .venv\Scripts\python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```
- **权限**：不要将项目放在需要管理员权限的目录（如 `C:\Program Files`）；或对当前用户给该文件夹「修改」权限。
- **杀毒/安全软件**：临时关闭可能拦截 pip 或 Python 的实时保护，安装完成后再开启。

### 3. 依赖安装时报「No matching distribution」或版本错误

- 本工具需要 **Python 3.8+**。在 cmd 中执行 `python --version` 确认版本。
- 若本机存在多个 Python（如 Anaconda、Store 版、官网版），可能用错了解释器。在项目目录执行：
  ```text
  where python
  ```
  确认第一个路径是你刚安装的 Python，必要时用完整路径，例如：
  ```text
  "C:\Users\你的用户名\AppData\Local\Programs\Python\Python312\python.exe" -m pip install -r requirements.txt
  ```

### 4. 启动应用后浏览器打不开或提示无法访问

- 确认黑框窗口没有报错且一直保持打开。
- 手动在浏览器输入：**http://localhost:8501**
- 若提示端口被占用，可先关闭其他可能占用 8501 的程序，或在本工具后续版本中改用其他端口（若有配置项）。

### 5. 双击 .bat 后窗口一闪而过

- **原因**：多半是 Python 未找到或依赖缺失，脚本报错后直接退出。
- **处理**：在资源管理器中按住 Shift，右键「在此处打开 PowerShell 窗口」或「在终端中打开」，然后手动执行：
  ```text
  .\启动应用.bat
  ```
  或：
  ```text
  python --version
  python -m venv .venv
  .venv\Scripts\python -m pip install -r requirements.txt
  .venv\Scripts\python -m streamlit run app.py
  ```
  这样可以看到完整报错信息，再根据提示排查（通常是 Python 未装或未加入 PATH）。

## 六、安装与启动流程小结

| 步骤 | 操作 |
|------|------|
| 1 | 安装 Python 3.8+（用项目内 `python` 文件夹的安装包或 python.org），**勾选 Add to PATH** |
| 2 | 关闭并重开命令行，执行 `python --version` 验证 |
| 3 | 进入「UV-Vis 光谱导数分析工具」目录，双击 **`启动应用.bat`**（首次会自动建 `.venv` 并安装依赖） |
| 4 | （可选）若希望提前装好依赖，再执行 **`安装依赖.bat`** |

日常使用只需：进入项目目录 → 双击 **`启动应用.bat`**。
