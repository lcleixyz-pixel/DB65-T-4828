# 光谱导数分析工具

对紫外–可见光谱仪导出的谱图进行本地分析：平滑、基线校正、一/二阶导数、极值检测与多视图可视化。适用于宝石鉴定、和田玉皮色鉴别等。

## 快速开始（Windows）

1. 双击 `启动应用.bat`。
2. 首次运行会自动完成以下动作：
   - 创建项目本地虚拟环境 `.venv`
   - 安装 `requirements.txt` 中的依赖
   - 启动应用并打开 `http://localhost:8501`
3. 之后再次双击 `启动应用.bat` 即可直接使用。

如果只想提前安装环境，可先双击 `安装依赖.bat`，再启动应用。

## 命令行启动（可选）

```bash
python -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt
.venv\Scripts\python -m streamlit run app.py --server.port 8501
```
