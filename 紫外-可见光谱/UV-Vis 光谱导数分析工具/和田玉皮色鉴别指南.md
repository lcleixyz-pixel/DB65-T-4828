# 和田玉皮色漫反射光谱（DRS）鉴别指南

基于光谱导数分析的皮色真伪及成因判定方法。通过一阶导数光谱的 435 nm / 535 nm / 575 nm 特征峰，鉴别天然针铁矿/赤铁矿致色与人工染色/烧红处理。

## 1. 原理与适用范围

### 1.1 核心机制
漫反射光谱法捕捉皮色中**致色矿物的电子跃迁指纹**，一阶/二阶导数可放大微弱特征峰、抑制背景干扰，实现无损鉴别。

- **物质专一性**：针铁矿、赤铁矿及人工铁黄/铁红具有不同吸收峰位
- **定量潜力**：峰强度与致色矿物含量相关

### 1.2 适用范围
- ✅ 和田玉子料皮色（黄皮、红皮、洒金皮等）
- ✅ 翡翠红翡/黄翡次生色、南红玛瑙等铁质致色宝玉石
- ⚠️ 不适用于有机宝石（如琥珀、珊瑚）

## 2. 关键波长指纹库

### 2.1 天然黄皮（针铁矿 Goethite, α-FeOOH）
| 参数 | 数值 | 诊断意义 |
|------|------|----------|
| **主峰** | **535 nm** | Fe³⁺ d-d 跃迁 |
| **诊断性次峰** | **435 nm** | 判定天然针铁矿的**关键依据** |
| **峰形** | 双峰（W 型/M 型） | 主峰 + 次峰缺一不可 |
| **峰宽** | 半高宽约 30–50 nm | 天然矿物结晶不均 |

435 nm 次峰强度约为主峰的 30%–60%；主峰可能随含量在 507–535 nm 间小幅移动。

### 2.2 天然红皮（赤铁矿 Hematite, α-Fe₂O₃）
| 参数 | 数值 | 诊断意义 |
|------|------|----------|
| **主峰** | **560–575 nm**（典型 575 nm） | Fe³⁺ 电荷转移 |
| **次级特征** | 微弱 435 nm 残余 | 混合针铁矿（天然过渡） |
| **峰形** | 单峰、相对尖锐 | 半高宽约 20–40 nm |

天然红皮常保留 435 nm 微弱信号；若 435 nm 完全消失且 575 nm 极强，需警惕烧红。

### 2.3 人工处理皮色特征

**有机染料**：无 435/535/575 nm 特征峰；宽缓吸收或异常峰位（如 496 nm）；二阶导数杂乱。

**人工铁黄（Fe₂O₃·H₂O）**：主峰**红移**至 **543–546 nm**（偏离 535 nm）；435 nm 次峰弱化或消失；峰形更尖锐对称。

**人工铁红（α-Fe₂O₃）**：主峰约 574 nm，与天然 575 nm 极近，需结合显微观察（染料在裂隙中堆积）等辅助判据。

**热处理（烧红）**：575 nm 峰**极强**；435 nm 峰**完全消失**；峰宽更窄（高温结晶）。机理为针铁矿加热脱水转化为赤铁矿。

## 3. 快速操作流程（5 步）

### 步骤 1：启动与上传
双击 `启动应用.bat`，在侧边栏点击「上传数据文件」，选择漫反射光谱的 `.txt` 或 `.csv`（第一列波长 nm，第二列反射率 0–1 或 0–100%）。

### 步骤 2：转换为 K–M 函数（关键）
1. 主界面展开 **「🔄 数据类型转换」**
2. **当前类型**：反射率(0–1) 或 反射率(%)
3. **转换为**：Kubelka–Munk
4. 点击 **「执行转换」**

K–M 函数 F(R) = (1−R)²/(2R) 将反射率转为吸收/散射比，使吸收峰更明显，是漫反射分析的标准做法。

### 步骤 3：加载预设并设置视图
1. 侧边栏「参数预设」下拉选 **「和田玉K-M标准」**，点击「加载」
2. 「视图模式」选 **「单图放大」**
3. 主界面图表上方选 **「一阶导数」**

噪声大时可选「和田玉K-M-降噪」；需更细峰形时可选「和田玉K-M-高分辨」。

### 步骤 4：添加参考线
在侧边栏「📍 标记」→「参考线」区域：
- 点击 **「🟡 快速添加和田玉参考线」**，可一次添加 435 nm（针铁矿次峰）、535 nm（针铁矿主峰）、575 nm（赤铁矿峰）三条标准参考线
- 或手动输入波长、标签、颜色后点击「➕ 添加参考线」

### 步骤 5：观察与判定
在一阶导数图中观察 400–600 nm：
- **435 nm 是否存在**（天然黄皮/天然红皮残余的关键）
- **主峰位置**：535 nm（天然黄皮）还是 543–546 nm（人工铁黄）
- **红皮样品**：575 nm 是否极强且 435 nm 完全消失（烧红）

可点击图上峰顶自动填入坐标，在「📍 标记」中加说明后「➕ 添加标记」。分析完成后在「💾 导出」下载 Excel、PNG 或 HTML。

## 4. 判定标准与决策树

### 4.1 决策流程
```
采集漫反射光谱 (R vs λ)
    → 转换为 K–M 函数 F(R)
    → 计算一阶导数 dF/dλ
    → 检查 435 nm 峰
        ├ 存在且明显 → 再看主峰：535 nm → 天然黄皮；红移 543–546 nm → 人工铁黄
        └ 不存在/极弱 → 再看主峰：575 nm 极强 → 烧红；其他峰位/宽缓 → 有机染料或人工铁红
```

### 4.2 判定标准矩阵
| 检测项 | 天然黄皮 | 天然红皮 | 人工铁黄 | 人工铁红 | 烧红 | 有机染料 |
|--------|---------|---------|---------|---------|------|---------|
| **435 nm 次峰** | ✅ 强 | ⚠️ 弱 | ❌ 很弱/无 | ❌ 无 | ❌ 无 | ❌ 无 |
| **535 nm 峰** | ✅ 主峰 | ❌ 无 | ⚠️ 红移至 543–546 | ❌ 无 | ❌ 无 | ❌ 无 |
| **575 nm 峰** | ❌ 无 | ✅ 主峰 | ❌ 无 | ✅ 主峰 | ✅ 极强 | ❌ 无 |
| **峰形** | W 型双峰 | 单峰 | 变形双峰 | 单峰 | 尖锐单峰 | 宽缓杂乱 |

### 4.3 典型判定结论
- **天然黄皮**：434–435 nm 强次峰 + 524–535 nm 主峰，W 型双峰 → 天然针铁矿致色
- **天然红皮**：575 nm 主峰 + 435 nm 微弱残余 → 天然赤铁矿（可能混少量针铁矿）
- **人工铁黄**：435 nm 弱/无 + 主峰 543–546 nm → 人工氧化铁黄染色
- **烧红**：575 nm 极强 + 435 nm 完全消失 → 热处理（烧红）
- **有机染料**：无铁矿物特征峰或异常峰位（如 496 nm）、峰形宽缓 → 有机染料作伪

## 5. 典型案例与图谱特征

### 5.1 天然针铁矿双峰
一阶导数呈 M 型或 W 型：400–435 nm 次峰、435–535 nm 主峰；435 nm 峰高约为主峰 30%–60%。

### 5.2 人工铁黄主峰红移
435 nm 次峰消失或极弱；主峰在 543–546 nm，峰形更对称尖锐。

### 5.3 烧红纯赤铁矿
575 nm 单峰极强；435 nm、520 nm 附近针铁矿峰完全消失；峰宽较窄。

### 5.4 有机染料
无 435/535/575 nm 特征峰；可能在 496 nm 等异常位置出现峰；整体宽缓圆滑。

## 6. 仪器与数据采集

### 6.1 推荐配置
- UV–Vis 漫反射光谱仪（DRS），波长 350–800 nm，分辨率 ≤ 2 nm（建议 ≤ 1 nm）
- 积分球、BaSO₄ 或 Spectralon 白板校准

### 6.2 样品与测量
- 表面清洁（如无水乙醇擦拭），选皮色均匀、无裂隙区域
- 同一样品至少 3 点测量取平均，测量前白板校准

## 7. 常见问题与注意事项

**Q1：为什么必须转 K–M？**  
反射率与吸收非线性；K–M 将其线性化，吸收峰更明显，便于导数分析。

**Q2：435 nm 附近有微弱信号，如何判断是否为针铁矿峰？**  
强度应 ≥ 主峰约 20%；峰位在 430–440 nm；峰形清晰非噪声。可适当增大一阶导数窗口（如 31–35）或使用「和田玉K-M-降噪」预设。

**Q3：人工铁红 574 nm 与天然 575 nm 难区分？**  
需结合显微观察（染料在裂隙中堆积）、紫外荧光、多点测量一致性等辅助手段。

**Q4：混合黄皮+红皮？**  
分区测量；若同一测点同时出现 435、535、575 nm 峰，多为针铁矿+赤铁矿过渡区，常对应天然氧化过渡。

**Q5：导数窗口怎么选？**  
一阶 25–31，二阶 51–71（约为一阶 1.5–2 倍）。窗口过大可能压掉 435 nm 次峰，可适当减小。

**Q6：基线校正后出现负值？**  
工具会自动裁剪并可能提示。建议对反射率先转 K–M 再基线校正；或适当降低基线 λ。

**Q7：能否只用二阶导数判定？**  
一阶导数为主判据（峰位清晰、文献对应）；二阶导数作辅助验证；二阶对噪声敏感，不建议单独作为判定依据。
